# CarConnective

**Connected Vehicle Intelligence Platform**

Website for CarConnective & FleetConnective — a connected vehicle intelligence platform for automotive dealerships and fleet operators.

## Stack

- Static HTML/CSS/JS
- Three.js (3D hero scene)
- GSAP + ScrollTrigger (animations)
- Deployed on Vercel

## Local Development

```bash
npx serve .
```

Then open `http://localhost:3000`

## Deployment

Push to `main` branch — Vercel auto-deploys.
